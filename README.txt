🌍 Travel Tracker

Essa aplicação fullstack voltada para o registro de países visitados, com visualização interativa em um mapa-múndi colorido. O principal objetivo do projeto é aprofundar o domínio prático de Node.js, Express.js e PostgreSQL, além de integrar dados geográficos com a experiência do usuário.

---

🚀 Objetivos do Projeto

- Desenvolver uma API RESTful com Express.js
- Trabalhar com banco de dados relacional usando PostgreSQL
- Registrar e armazenar países visitados pelo usuário
- Visualizar no mapa os países visitados com destaque em cores
- Praticar integração entre backend e visualização geográfica
- Estruturar um projeto limpo, modular e escalável



🛠️ Tecnologias Utilizadas

- [Node.js](https://nodejs.org/)
- [Express.js](https://expressjs.com/)
- [PostgreSQL](https://www.postgresql.org/)


🌐 Funcionalidade Principal

✅ O usuário pode registrar os países que já visitou  
✅ Cada país registrado é salvo no banco de dados  
✅ O sistema destaca em cor personalizada os países visitados no mapa  
🔜 Futuras funcionalidades: categorias de viagens, filtros por continente, dados por viagem, entre outros.
